# 왁뿌볼 파손 그래픽·ASMR 사운드 개선 설계

- 작성일: 2026-07-10
- 대상 저장소: `yeonjunim521-tech/wakppubol`
- 대상 브랜치: `master`

## 목표

터치할 때 왁스 외피가 단계적으로 갈라지고 무너지는 모습을 더 입체적이고 현실적으로 표현한다. 파손음은 CC0/Public Domain 실제 녹음 샘플과 Web Audio 합성을 혼합해 반복 재생의 인공적인 느낌을 줄인다.

## 범위

- 기존 7회 터치 후 내부 말랑이가 드러나는 게임 흐름 유지
- SVG 기반 균열·파편 레이어 추가
- CSS 기반 표면 질감, 단면 깊이, 파편 애니메이션 개선
- CC0/Public Domain 파열음 샘플 여러 개 로컬 저장
- 실제 샘플과 합성 저역·미세 균열음을 단계별로 혼합
- 음원 로딩 실패 시 합성음 자동 대체
- 모바일 터치, 키보드 조작, reduced-motion 접근성 유지
- 음원 출처와 라이선스 기록

## 제외 범위

- Canvas/WebGL 물리 엔진
- 서버, 사용자 계정, 분석 기능
- 원본 YouTube 영상의 음원 추출 또는 복제
- CC BY 등 출처 표시가 의무인 음원

## 그래픽 설계

### 표면

기존 원형 왁스 레이어 위에 불규칙한 미세 요철, 뽀얀 분말, 반투명 색층을 추가한다. 유광 플라스틱 느낌을 줄이고 얇고 부서지기 쉬운 왁스 껍질로 보이게 한다.

### 균열 진행

1~2회는 터치 중심에서 시작하는 가는 균열과 작은 백화 흔적을 표시한다. 3~4회는 가지 균열과 작은 들뜸을 확장한다. 5~6회는 어두운 틈, 밝은 파단면, 일부 떨어진 조각을 노출한다. 7회는 외피가 여러 불규칙 파편으로 벌어지며 내부 말랑이가 드러난다.

균열은 SVG path로 정의하고 상태별로 stroke-dashoffset, opacity, filter를 변경한다. 파편은 SVG polygon 또는 기존 DOM 조각을 사용하며 터치마다 짧은 이동·회전 애니메이션을 적용한다.

### 터치 위치

Pointer Event의 좌표를 왁뿌볼 내부 비율로 계산해 CSS custom property로 전달한다. 충격 하이라이트와 미세 파편의 시작점을 실제 터치 위치에 가깝게 표시한다. 게임 진행 수와 핵심 균열 구조는 결정적으로 유지해 어느 위치를 눌러도 최종 상태가 안정적으로 보이게 한다.

## 사운드 설계

### 음원 정책

CC0 또는 명확한 Public Domain 음원만 사용한다. 각 파일의 원본 URL, 제작자 표기, 확인한 라이선스, 확인 날짜를 `assets/audio/LICENSES.md`에 기록한다. 라이선스가 불명확한 파일은 사용하지 않는다.

### 재생 구성

- 초기 균열: 짧고 마른 실제 파열 샘플 + 약한 고주파 합성 크랙
- 중간 균열: 낮은 피치의 파열 샘플 + 얇은 파편 연속음
- 최종 파손: 강한 실제 파열 샘플 + 짧은 저역 바디 + 여러 미세 크랙
- 말랑이: 기존 합성음을 유지하되 볼륨과 필터를 부드럽게 조정

같은 상태에서도 샘플, playbackRate, gain을 작은 범위에서 무작위화한다. 과도한 볼륨 중첩을 막기 위해 master gain과 compressor를 사용한다.

### 오류 처리

페이지 초기화 시 음원을 비동기로 준비한다. 첫 상호작용 전 브라우저 자동재생 정책을 존중하며, 파일 로딩·디코딩에 실패하면 해당 이벤트는 기존 합성음만 재생한다. 음원 실패가 게임 입력이나 그래픽 진행을 막지 않는다.

## 구조

- `index.html`: SVG 균열·파편 레이어와 접근성용 마크업
- `styles.css`: 표면 질감, 상태별 균열·파편, 터치 반응 애니메이션
- `app.js`: 터치 좌표, 상태 전이, 샘플 로더, 혼합 오디오 엔진
- `assets/audio/`: CC0/Public Domain 오디오 파일
- `assets/audio/LICENSES.md`: 출처와 라이선스 증빙
- 테스트 파일: 상태 전이, 사운드 선택, 로딩 실패 fallback을 검증하는 최소 테스트

## 검증 기준

- 1~7회 터치 단계가 시각적으로 구분됨
- 최종 파손에서 외피 파편과 내부 말랑이가 명확히 보임
- 동일 단계 반복 재생 시 소리가 기계적으로 동일하지 않음
- 네트워크를 끊어도 로컬 음원이 정상 재생됨
- 음원 파일을 누락시켜도 합성음 fallback으로 게임이 동작함
- 모바일 화면에서 터치 중 이중 입력이 발생하지 않음
- 키보드 Enter/Space 조작과 reduced-motion 설정이 유지됨
- 사용된 모든 외부 음원이 CC0/Public Domain으로 문서화됨

## 리스크와 대응

- CC0에서 정확한 왁스 파열음을 찾기 어려울 수 있음: 얇은 플라스틱·달걀껍질·마른 잎 파열 등 재질이 유사한 샘플을 짧게 편집하고 합성층으로 보정한다.
- 오디오 파일 증가로 초기 로딩이 느려질 수 있음: 짧은 mono 파일로 압축하고 필요한 샘플만 사전 로드한다.
- SVG 레이어가 복잡해질 수 있음: 균열과 파편을 독립 그룹으로 분리하고 상태별 data attribute만 사용한다.
